
# Repo Remover

GitHub 레포지토리를 한 번에 관리하고 삭제하는 웹 애플리케이션입니다.

## 이런 분들께 유용해요

- 😅 불필요한 레포지토리로 정리가 필요한 개발자
- 🧹 주기적으로 GitHub을 청소하고 싶은 분
- 📊 전체 레포지토리 현황을 빠르게 확인하고 싶은 분

## 주요 기능

- 🔐 GitHub 계정으로 안전하게 로그인
- 📋 내 모든 레포지토리 목록 확인
- ✅ 여러 레포지토리 동시 선택
- 🗑️ 선택한 레포지토리 한 번에 삭제
- 🌙 다크/라이트 모드 자동 지원

## 사용법

1. **GitHub 연동** - "Connect with GitHub" 버튼 클릭
2. **레포지토리 선택** - 삭제할 레포지토리 선택 버튼 클릭
3. **일괄 삭제** - "선택한 레포지토리 삭제"로 한 번에 정리

---

⚠️ **주의**: 삭제된 레포지토리는 복구할 수 없으니 신중하게 선택하세요.
